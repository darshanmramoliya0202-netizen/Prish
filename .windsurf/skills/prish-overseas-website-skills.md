- Translate Indian agricultural heritage into a modern digital brand system
- Convert brochure facts into refined global-facing website copy
- Balance emotional storytelling with B2B credibility

## UX Strategy Skills
- Design for importers, distributors, formulation teams, and brand owners
- Build inquiry-first journeys with clear calls to action
- Structure pages for trust, clarity, and conversion

- Create a premium web identity distinct from print brochure aesthetics
- Use typography, spacing, depth, contrast, and motion to create memorability
- Express the spirit of India in a subtle and contemporary way

## Frontend Build Skills
- Build with a modern component-based architecture
- Deliver responsive layouts across desktop, tablet, and mobile
- Use motion and layered visuals without harming performance
- Maintain accessible contrast and semantic structure

## Content System Skills

- Surface certifications, compliance, packaging, and export readiness clearly
- Present technical and commercial information in a premium, digestible way

## Quality Control Skills
- Keep every feature aligned with the approved brand rules
- Prevent brochure replication in layout or visual composition
- Track all implementation changes in `CHANGE_LOG.md`
